YMH 216 - Yazılım Ekonomisi


Hafta-2

  • Esneklik = değişim(%) / Fiyat değişimi(%)

  • kâr = toplam gelir - toplam maliyet

  • Toplam gider = arz + emek + fırsat maliyeti

  • Katma Değer = p - D
    • Bir ürünün birim satış fiyatı (p) ile toplam birim maliyeti (D) arasındaki farktır

  • Üretkenlik = Çıktı / Girdi

  • Karlılık=Toplam kâr / Toplam sermaye
    • Not: kârlılık ile kâr farklı kavramlardır.
